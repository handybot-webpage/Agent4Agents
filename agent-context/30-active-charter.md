# Charter — readme

**Tier:** 1
**Goal:** Write an extremely high-quality `README.md` — the project's front door.

## In-scope
- `README.md`

## Out-of-scope
- everything else

## Linked ADRs
- ADR-0001 — Markdown is the data layer
- ADR-0004 — Integrate gstack as the workflow engine

## Acceptance evidence
- Required sections present: pitch, problem, solution, quick-start (new + existing), architecture, demonstrated working, augmentation summary, performance, status, FAQ, contributing, license, references.
- Length 250–500 lines.
- All internal links resolve.
- Honest performance range (20–100×, not 1000×).
